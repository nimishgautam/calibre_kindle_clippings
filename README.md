Based roughly off the calibre-annotations extension, this is just a 'standalone' script that can be run via the calibre-debug console to import a 'My Annotations' file from a kindle.

I created this because I value my annotations/highlights/notes etc in my kindle books quite highly, and the existing annotations extension didn't go granular enough for me. This also does it via cli with a lot of interactive decisions and saved/copied DB so chances of things going wrong are less.
